EYO Privacy Policy

Last updated: 2026-04-26

1. Overview

EYO is a time management app for independent professionals. It helps users create projects, track time, add manual entries, review reports, and export time data when needed.

2. Local-First Data Storage

EYO stores app data locally on the user's device by default. This may include projects, client names entered by the user, hourly rates, time entries, start and end times, notes, billable and non-billable status, reports generated from local entries, app settings, and Pro status.

By default, this data is not uploaded to an EYO server.

3. No Account Required

EYO can be used without creating an account. The current version does not require user registration, login credentials, email-based account creation, or cloud profile creation.

4. Data Collection

In the current version, EYO does not intentionally collect personal data on an external server. Data entered into the app remains stored locally on the device unless the user chooses to export or share it.

5. Data Sharing

EYO does not share user data with third parties by default. The user may choose to export or share data manually, for example by creating a CSV export and using the device share sheet.

6. CSV Export

EYO may allow users to export time tracking data as CSV. A CSV export may include project names, time entry dates, start and end times, durations, notes, billable status, and calculated values. CSV files are only created when the user chooses to export or share them.

7. Pro Access

Some EYO features may be limited to Pro users. During the first release or testing phase, Pro access may be enabled through an unlock code. Future versions may use official in-app purchases or subscriptions.

8. Cloud Sync and Backup

The current version does not include cloud sync by default. The current version may not include automatic backup or restore features. Users are responsible for keeping any exported files safe.

9. Device Permissions

EYO may use standard device features when required by app functionality, such as sharing exported files through the operating system share sheet. EYO does not request unnecessary permissions for core time tracking functionality.

10. Data Deletion

Because EYO stores data locally by default, users can delete app data by deleting entries, projects where supported, or removing the app and its local data from the device.

11. Children

EYO is not designed for children and is intended for productivity and professional use.

12. Changes to This Policy

This Privacy Policy may be updated as EYO evolves. Changes may be required when new features are added, such as cloud sync, account creation, analytics, or official subscriptions.

13. Contact

For support, feedback, or privacy-related questions, contact: eyo.app.support@gmail.com
